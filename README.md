# ByteLie
This is a simple cybersecurity tool which uncovers real file exetensions.Does not matter if extenssion is faked or hacker malformed it or wheteever.just give it to ByteLie to eat it.stay safe!

## How to Use?
- clone this git repository in directory of your choice with git clone
- get into this directory with cd
- finaly execute filechecker.py with python3 filechecker.py
- enter file path which you want to test and taht's it,game over :)
